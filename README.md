# EventGrid

EventGrid is a web-based university event management system that allows users to explore and register for events while enabling administrators to create and manage event listings efficiently.


*Features

- Role based access control
- Event browsing
- View detailed event information
- Provide event related notifications
- User event registration as participant, volunteer
- Admin panel to add, update and delete events
- Database integration

*Technologies Used

- HTML, CSS
- PHP
- MySQL

*Purpose

This project was developed as part of a coursework assignment to demonstrate full-stack web development skills including frontend design, backend logic and database integration.
